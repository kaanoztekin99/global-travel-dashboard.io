# Global Travel Dashboard

This project is prepared for GitHub Pages deployment.

## GitHub Pages setup

- The site content is placed in the `docs/` folder.
- Enable GitHub Pages in repository settings using the `main` branch and `docs/` folder.
- If you push to a remote repository, the site should become available at `https://<username>.github.io/<repo-name>/`.

## Local preview

Open `docs/index.html` in a browser or use a local static server.
